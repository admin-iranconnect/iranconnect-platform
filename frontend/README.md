IranConnect - Frontend (Final)

This project is a Next.js (Pages Router) frontend styled for the IranConnect project.

Features:
- Colors: 60% white (page background), 20% navy, 20% turquoise
- Header (white), Body (white), Footer (navy) separated
- All texts in English
- Logo should be placed at /public/logo.png (500x500 recommended) and will be displayed at header and fallback image

Quick start:
1. cd frontend folder
2. Install: npm install
3. Create .env.local with:
   NEXT_PUBLIC_API_BASE=http://localhost:5000
4. Run: npm run dev

Replace /public/logo.png with your actual logo (500x500) to display in the header.
